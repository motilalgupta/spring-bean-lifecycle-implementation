# Spring Bean Lifecycle Implementation

This project demonstrates how the Spring IoC Container manages the complete lifecycle of a bean from creation to destruction.

## Concepts Covered

* Spring IoC Container
* Bean Lifecycle
* Bean Creation
* Dependency Injection
* Initialization Methods
* Destroy Methods

## Lifecycle Flow

1. Bean Instantiation (Constructor Execution)
2. Dependency Injection
3. Initialization Method Execution
4. Business Logic Execution
5. Destroy Method Execution

Example Output:

```text
Motor Constructor...
Motor started...
Motor is pumping...
Motor stopped...
```

## Project Structure

* Motor Bean
* XML Configuration
* Init Method
* Destroy Method
* Test Class

## Learning Outcomes

* Understand the role of the Spring IoC Container.
* Learn how bean initialization works.
* Learn how bean destruction works.
* Understand the complete lifecycle of a Spring Bean.
